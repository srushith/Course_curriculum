Course Modules: A Guided Path

Week 1: Python Fundamentals Crash Course

Build a robust Python foundation for AI development.

  • Introduction to Python & Google Colab
  • Variables, Data Types (Lists, Tuples, Sets, Dictionaries)
  • Operators, Conditionals, Loops
  • Functions, Object-Oriented Programming (Classes)
  • Modules & Packages, Virtual Environments

Week 2: Python for Gen AI

Essential libraries for data manipulation and visualization.

  • NumPy: Arrays, Indexing, Slicing, Broadcasting
  • Pandas: DataFrames, Filtering, Missing Values, Manipulation
  • Data Analysis with Groupby & Merge
  • Matplotlib: Line, Scatter, Bar, Pie Charts & Heatmaps

Week 3: Hands-on with Generative AI

First steps into the world of LLMs and Diffusion Models.

  • Language Models Overview & Text Generation
  • LLMs in Practice: API & Open/Closed Models
  • Prompt Engineering: Patterns & Best Practices
  • Diffusion Models: Text-to-Image Generation (DALL-E)

Week 4: Deep Dive into LLMs

Understanding the architecture and capabilities of LLMs.

  • Language Modeling: N-Grams & Limitations
  • Intro to LLMs: MMLU, Tokens, Embeddings
  • Transformers: Architecture, Self-Attention, Decoding
  • Positional Encoding, LLM Training Overview

Week 5: Building Applications with LLMs

Develop practical applications using advanced techniques.

  • Retrieval Augmented Generation (RAG): Embedding, Chunking
  • LangChain: Introduction & Demo
  • LLM Agents: Tools, Memory, Planning
  • LLM Safety: Data Exfiltration

Week 6: Training LLMs

Techniques for customizing and optimizing LLMs.

  • Pre-training LLMs: Methodologies & Stages
  • Fine-Tuning LLMs: Types, Procedure, Best Practices
  • LoRA & Instruction Tuning
  • Reinforcement Learning from Human Feedback (RLHF)

  • Develop chat-based app with Streamlit & LangChain
  • Generate dashboard visualizations

Week 8: Financial Bot with LLMs

  • Web scraping for financial data
  • Integrate LLMs with VectorDB (Chroma) for RAG
  • Build responsive web interface with Streamlit

Build a voice-activated bot for real-time conversations.

  • Speech recognition with OpenAI Whisper

  • Interactive UI with Streamlit & conversation memory.
